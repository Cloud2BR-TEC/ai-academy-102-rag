# Building a Multi-Agent RAG Chatbot

This site documents two practical options for building a Multi-Agent RAG sample and explains shared architecture, setup, and publishing flow.

## Learning Goals

- Understand what Multi-Agent RAG is and when to use it.
- Compare two implementation styles for the same concept.
- Reuse shared repositories to create a publishable docs site.
- Keep preview-first links with optional download actions.
